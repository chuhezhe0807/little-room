关于我

01

基本信息

掌握HTML5 + CSS3，能结合 Sass 等预处理器进行设计稿还原
掌握JavaScript并能运用 ES6+ 语法进行高效开发
熟练使用Vue全家桶、React全家桶相关技术 会使用Three.js进行开发工作
理解Webpack、Vite前端构建工具，实现过简单的Loader、Plugin
了解数据结构与算法
GitHub: https://github.com/chuhezhe0807

个人项目

1、技术栈: Three.js + Blender 个人介绍项目（当前项目）
2、技术栈: React + Redux + TypeScript + 原生Canvas + Ant Design 一个常见的通用后台管理系统，封装Axios，动态路由，使用Redux作为状态管理工具。canvas部分实现画布放缩、拖拽，支持画布中元素的修改大小、移动位置和旋转。canvas原生实现动画，使用A*算法实现自动寻路、绘制生长梅花图案等。
3、技术栈: Node.js 根据配置文件自动生成路由表的命令行工具。深度优先遍历、对象序列化写入文件。

其他技能

1、会使用Blender构建三维模型。
2、熟练使用World、Excel和PPT等办公软件，会使用Visio梳理业务，绘制业务流程。
